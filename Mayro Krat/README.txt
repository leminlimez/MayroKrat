-----------------------------------------------------------
Running the Project
-----------------------------------------------------------
To run the project, simply run Main.py. It takes around 10 seconds to compile.

Numba, Numpy, and PyGame modules required!

Recommended speed is 100 CC.

-----------------------------------------------------------
Controls
-----------------------------------------------------------
W - Accelerate
S - Decelerate
A - Turn Left
D - Turn Right

Left Shift - Tap to hop, Hold to drift
Space - Use Item

-----------------------------------------------------------
Modes & Speeds
-----------------------------------------------------------
--// Time Trials \\--
Race for the best time on each track. Use mushrooms to take shortcuts.

--// Speeds \\--
50 cc - Slowest (Easiest)
100 cc - Default speed
150 cc - Fastest speed (Most difficult)
